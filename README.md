## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/devallabharath/myautolog.git
   ```
2. Install NPM packages
   ```sh
   npm install
   or
   yarn
   ```

## Usage

1. Clone the repo
   ```sh
   npm run start
   or
   yarn start
   ```
2. Open `localhost:3000` in your browser
